One Social Feed

A single, chronological feed for all your social media.

One is a web application that aggregates several social media feeds in chronological order. - Streamlined three-legged oAuth process through Flask/Python Server route logic - Architected API calls to integrate SoundCloud and Instagram embeds on front end - Constructed interactive data visualizations using D3 to present social feed intelligence - Standardized date & time results from social media APIs for UI filtering in descending order
Built with: Python / Flask, MySQL / SQLAlchemy, Angular JS, Twitter API, Instagram API, Soundcloud API, Reddit API
